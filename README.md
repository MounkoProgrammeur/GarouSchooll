# GarouSchooll

**GarouSchooll** est une application web de gestion administrative universitaire, conçue pour simplifier et centraliser les interactions entre étudiants, enseignants et administration.

## Fonctionnalités

### Espace Étudiant
- Tableau de bord personnalisé avec suivi des performances
- Consultation des notes et relevés
- Suivi des paiements (avec système de tranches)
- Accès aux documents administratifs
- Notifications et annonces en temps réel
- Gestion du profil et des paramètres

### Espace Enseignant
- Tableau de bord enseignant
- Gestion des notes
- Suivi de présence (attendance)
- Diffusion d'annonces
- Paramètres du compte

## Stack technique

- **Backend** : Django (Python)
- **Frontend** : Bootstrap 5, CSS personnalisé, JavaScript
- **Visualisation** : Chart.js (graphiques de performance)
- **Icônes** : Bootstrap Icons
